1. Instalar fastapi 
```
conda install -c conda-forge fastapi
```

2. Instalar uvicorn para implementar el servidor
```
conda install -c conda-forge uvicorn
```

3. Crear archivo main.py 

4. Ejecutar como
```
uvicorn main:app
```
4. Ejecutar como un demonio
```
uvicorn main:app --reload
```
